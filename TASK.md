# Task: VBT Showcase 웹페이지

## 목적
Roomfit AI-VBT 리포트의 인터랙티브 데모 웹페이지.
마케터가 링크만 열면 "이 제품이 뭘 보여주는지" 바로 이해할 수 있는 쇼케이스.

## 참고 파일
- `metric_data_reference.dart` — 12개 VBT 지표 설명 데이터 (한국어). 이 내용을 그대로 웹에 구현.

## 요구사항

### 페이지 구조 (Single Page, 탭 전환)
1. **세트 리포트** 탭 — 1세트 운동 결과 데모
2. **세션 리포트** 탭 — 3세트 전체 세션 결과 데모
3. **지표 가이드** 탭 — 12개 지표 카드형 레이아웃, 트레이너 설명 멘트 포함

### 세트 리포트 (데모 데이터)
- 운동: 바벨 벤치프레스, Set 1, 무게 60kg, 8렙
- 헤더: 평균 속도(0.65 m/s), 렙 수(8), 무게(60kg)
- 6 메트릭 그리드: 평균 파워(320W), ROM(450mm), TUT(4.2s), Vel Loss(18%), L/R 밸런스(92%), 추정 1RM(76kg)
- Velocity Loss 차트 (바 차트, 렙별 속도 하강)
- L/R 밸런스 바
- Con/Ecc 속도 차트
- Bad Rep 피드백 (렙 6: 좌우 불균형, 렙 7: ROM 부족)
- AI 코치 분석
- Power Score: 49 (STRONG 💪)

### 세션 리포트 (데모 데이터)
- 3세트 요약: 평균 속도, 파워, 총 볼륨
- 세트별 비교 테이블
- 피로도 추이 차트 (선 그래프)
- AI 세션 코칭

### 지표 가이드
- 12개 지표를 카드형으로
- 각 카드에: 이모지, 타이틀, 한 줄 요약, "자세히 보기" 클릭 → 모달
- 모달: 무엇을 측정하나요? / 왜 중요한가요? / 어떻게 읽나요? / 💬 고객 설명 멘트

### 공통
- **클릭으로 지표 설명 모달** — 세트/세션 리포트에서도 지표 클릭 시 모달
- **다크/라이트 토글** — 우상단 토글
- **반응형** — PC/모바일
- **브랜드**: Primary #5252FA, Accent #BAFC27, 다크 배경 #0A0A0F
- **폰트**: system-ui 또는 Pretendard (CDN)
- **순수 HTML/CSS/JS** — 프레임워크 없음
- 한국어 UI

### 배포
- 파일: index.html (단일 파일, inline CSS/JS)
- Cloudflare Pages: `roomfit-vbt-showcase` 프로젝트
